### **1. Bleeding-Edge Tech Stack**

* **React 19 & Vite 7**: Leveraging the latest concurrent rendering features and lightning-fast HMR (Hot Module Replacement).

### **2. The Work History Engine (`jobRandomizer`)**

Rather than a static list, the home page features a custom-built randomization engine.

* **Logic**: A TypeScript-driven algorithm that pulls from an array of 22+ diverse professional roles (e.g., **Software Engineer Coach**, **Glass Blower**, **Santa**, and **Tuba Player**).
* **State Management**: Uses React `useState` to pair randomized roles with witty, context-aware commentary.

### **3. Responsive Glassmorphism & Motion UI**

* **Neo-Glass Aesthetic**: Achieved via custom CSS3 `backdrop-filter` and RGBA alpha-blending for a futuristic, translucent look.
* **Advanced Motion**: Orchestrated with **Framer Motion** for fluid, physics-based transitions and layout animations.
* **Data & Content**: Dynamic rendering of markdown via `react-markdown` and visual storytelling through **Recharts** data visualizations.
* **Zero-Dependency Layout**: High-performance grid and flex layouts built without bulky UI libraries to maximize Lighthouse scores.
